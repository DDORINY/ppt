Project Overview

도로 위 위험을 더 빠르게 발견하고 대응하기 위한 프로젝트

도로 낙하물은 교통 흐름을 방해하고 대형 사고로 이어질 수 있는 위험 요소입니다. 기존에는 시민 신고와 CCTV 수동 모니터링에 크게 의존했기 때문에 즉각적 대응이 어렵고, 그 사이 2차 사고 가능성이 높아지는 문제가 있었습니다.

기존 문제

  • 시민 신고 또는 수동 관제 중심
  • 탐지부터 판단까지 시간이 오래 걸림
  • 현장 상황 공유가 늦어짐
  • 2차 사고로 이어질 가능성 존재
→

해결 방향

  • 이미지·영상 기반 도로 낙하물 자동 탐지
  • AI 분석 결과의 구조화 저장
  • 관리자 페이지 실시간 알림 연동
  • 비교분석과 예방 확장까지 가능한 구조 설계

Project Goals

탐지에서 끝나지 않는 운영형 AI 시스템 구축

이번 프로젝트의 목표는 단순히 객체를 탐지하는 모델을 만드는 것이 아니라, 탐지 결과가 저장되고 관리자 대응으로 이어지며 실제 서비스처럼 동작하는 시스템을 만드는 것이었습니다.

Goal 01

실시간 객체 탐지

도로 낙하물을 이미지와 영상에서 탐지할 수 있는 AI 모델을 구현합니다.

Goal 02

신고-분석 연계

사용자의 업로드 데이터가 AI 분석으로 자동 연결되도록 구성합니다.

Goal 03

관리자 대응 체계

실시간 알림과 상태 관리가 가능한 관리자 화면을 구축합니다.

Goal 04

비교분석 기반 판단

여러 모델을 비교하며 운영 기준을 제시할 수 있는 구조를 만듭니다.

Needs Assessment

프로젝트가 필요한 이유를 수요 관점에서 검토

단순한 아이디어 제안이 아니라, 실제 사고 위험과 공공 활용 가능성을 고려해 시스템 필요성을 정리했습니다.

현실적 문제 상황

  • 고속도로와 일반도로에서 낙하물 사고가 반복적으로 발생
  • 현장 관제 인력만으로 모든 구간을 즉시 확인하기 어려움
  • 사고 발생 후 대응이 중심이 되어 예방적 관리가 부족함

공공 수요 관점

  • 도로 관리 기관의 실시간 모니터링 자동화 수요 증가
  • 위험 지역 통계화와 정책 지원 시스템 필요
  • 사고 예방 중심의 스마트 도로 인프라 확장 가능성

기획부터 발표까지 4주 일정

짧은 기간 안에 기획, 설계, 기능 구현, AI 모델 개발, 발표 준비까지 순차적으로 진행했습니다.

WEEK 3

데이터셋 정리, 학습, 비교분석, 결과 시각화

WEEK 4

Differentiators

LLM + GPS(EXIF)

이미지 메타데이터와 분석 정보를 연결해 신고 내용을 보조 자동화합니다.

02

실시간 위험 알림

03

탐지 현황 지도 시각화

낙하물 발생 위치를 지도로 시각화하여 현황 파악과 확장 기능의 기반을 만듭니다.

04

AI 비교분석

단일 모델 결과가 아니라 여러 모델을 비교해 운영 기준을 제시합니다.

Backend & DB Architecture

서비스 운영을 고려한 웹 시스템 구조 설계

Flask 기반 백엔드, MySQL 데이터베이스, 관리자 기능, AI 분석 이력을 중심으로 웹 서비스 구조를 설계했습니다.

기술 스택

  • Frontend: HTML, CSS, JavaScript
  • Backend: Flask, Flask-SQLAlchemy, Flask-Migrate
  • Database: MySQL, PyMySQL
  • AI: YOLOv8, RT-DETR, OpenCV
  • Realtime: Flask-SocketIO

핵심 테이블 구조

AI Application Flow

AI를 서비스 흐름에 녹여낸 적용 방식

AI는 단독 기능이 아니라 신고 등록, 저장, 알림, 비교분석, 관리자 대응까지 전체 서비스 흐름 안에 포함되어 작동합니다.

STEP 1

Input

사용자가 이미지 또는 영상을 업로드합니다.

STEP 2

Inference

AI 모델이 객체를 탐지하고 신뢰도를 계산합니다.

STEP 3

Persistence

탐지 결과와 상태 정보를 DB에 구조화 저장합니다.

STEP 4

Response

관리자 페이지에 알림과 분석 결과가 전달됩니다.

STEP 5

Expansion

비교분석과 통계 기반 예방 기능으로 확장됩니다.
